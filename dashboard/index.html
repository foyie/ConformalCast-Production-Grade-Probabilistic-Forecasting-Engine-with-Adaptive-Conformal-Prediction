MC DROPOUT EXPLAINED
why we use it
Training (standard)
Forward pass with dropout active. Learns robust features despite random neuron shutdowns.
Inference (MC Dropout)
Keep dropout ACTIVE. Run T forward passes. Get empirical distribution over T predictions.
Result
mean(T predictions) ≈ point forecast
std(T predictions) ≈ predictive uncertainty
percentiles(T) → prediction intervals

PARAMETER GUIDE
tuning EnbPI
window_size
Larger = more stable, slower to adapt. Default 720h (1 month).
alpha
Miscoverage target. 0.10 → 90% coverage. Lower = wider intervals.
batch_size
How many observations before recalibration. 24h = daily update.
